Computing Machines in World War 2



As the World War II began, the German created a very exceptional problem to the allies .It was called the Enigma Machine which exhibit different code each day, thus making it hard for the allies to break it. Cryptographers lead to the idea of developing a machine that can quickly do computation in a day to enable them to know what code does the enigma machine is using each day. Alan Manthis Turing who worked for the Government Code and Cypher school at Bletchley Park, Britain’s code breaking center, was the pioneer to the effort of making such computing machine. This computing machine was the now known as the “Computer”.
